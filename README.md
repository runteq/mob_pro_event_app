## 環境構築方法
### 他で動いているコンテナがある場合があるので止めておきましょう
※動作に関係無いことがわかっていれば止めなくとも構いません

```
docker stop $(docker ps -q)
```

### コンテナを立ち上げましょう
```
docker compose up
```
